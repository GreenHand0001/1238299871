# 项目简介

## 核心功能
- 基于 YOLOv8 的寄生虫检测系统，支持对8类寄生虫进行识别：钩虫属、蛔虫、蛲虫、肝片吸虫、膜壳绦虫、血吸虫、绦虫属、鞭虫
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供可视化界面，实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存功能
- 可自定义置信度和IOU阈值
- 可选择显示标签和原图
- 支持切换不同检测模型

## 适用场景
- 医学研究机构进行寄生虫种类识别
- 医学实验室的寄生虫检测分析
- 生物样本中的寄生虫自动筛查
- 医学教育中的寄生虫识别演示
- 需要快速识别寄生虫种类的科研场景

## 技术特点
- 基于 Python 3.9 开发，使用 OpenCV、PyQt5、Ultralytics、Torch1.9 等技术栈
- 提供 PyCharm+Anaconda 或 VSCode+Anaconda 两种开发环境支持
- 包含2110张寄生虫标注数据集，涵盖8类寄生虫
- 已训练好模型，配置环境后可直接使用
- 完整源码、数据集、训练结果及运行教程均包含在内
